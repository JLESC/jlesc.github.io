Research topic and goals

In this project, we aim at finding efficient fault-tolerant scheduling schemes for workflow applications that can be expressed as a directed acyclic graph (DAG) of tasks. A task can be checkpointed and/or replicated, so that if a single replica fails, no recovery is needed.

We propose a model of process replication for tasks, and a theoretical analysis allowing us to decide when to checkpoint/replicate a task.
