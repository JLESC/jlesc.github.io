This website is build with the following open source tools and libraries:

  • Jekyll with the following plugins
    • jekyll-assets
    • jekyll-scholar
    • jekyll-sitemap
    • jekyll-feed
    • jekyll-paginate
  • jQuery 2.1.4
  • Twitter Bootstrap v4-dev (at some commit between alpha.2 and alpha.3)
    • which depends on Tether.js for tooltips
  • minimal adjusted DataTables.js to work with Bootstrap v4-dev
    • with datetime plugin
      • which depends on moment.js
  • Featherlight.js for lightbox effects (e.g. on some images)
  • theme based on Cosmo Bootswatch
  • special fonts
    • font-awesome for icons
    • Roboto Font Family
  • deployment via Travis CI, GitHub Pages and jgd gem
